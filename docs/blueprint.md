# **App Name**: Nexus Launchpad

## Core Features:

- Student Profiles: Create and manage profiles showcasing skills, projects, and connections.
- AI Mentor Bot: Get personalized career guidance and project suggestions from an AI mentor tool.
- Global Collaboration Map: Explore an interactive map to find and connect with students worldwide, filtering by skills and online status.
- AI Resume Builder: Automatically generate resumes and portfolios from profile data with a single click export.
- Project Showcase: Enable students to present projects/startups and receive votes.
- College Hub: Provide a campus-specific page for each college within Nexus Launchpad where students can collaborate. It features posts/ideas limited to their school's community.

## Style Guidelines:

- Primary color: Electric Indigo (#6F00FF) to evoke innovation and a futuristic feel.
- Background color: Light gray (#F0F0F0) to ensure readability and a clean interface.
- Accent color: Cyan (#00FFFF) for interactive elements and highlights, creating a sense of technology.
- Font: 'Inter', a grotesque-style sans-serif, should be used for both headlines and body text, conveying a modern and objective look.
- Use minimalist line icons with a futuristic style.
- Clean and modern layout with clear sections, whitespace and intuitive navigation.
- Use subtle animations and transitions to enhance user experience without being distracting.
